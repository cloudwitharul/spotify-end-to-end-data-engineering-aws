# Data Warehouse Schema

Final curated Spotify dataset stored in Amazon S3 and queried using AWS Athena.

## Table: spotify_datawarehouse

| Column Name        | Description |
|--------------------|------------|
| track_id           | Spotify track unique identifier |
| track_name         | Track name |
| artist_id          | Artist unique identifier |
| artist_name        | Artist name |
| followers          | Artist followers count |
| artist_popularity  | Artist popularity score |
| album_id           | Album unique identifier |
| release_date       | Album release date |

## Storage Details
- Format: Parquet
- Compression: Snappy
- Partitioned output generated by AWS Glue
